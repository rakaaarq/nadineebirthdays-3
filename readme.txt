NADINE BIRTHDAY RAPUNZEL FINAL V2

Cara menjalankan:
1. Ekstrak ZIP ini.
2. Buka folder ini di Visual Studio Code.
3. Klik kanan index.html > Open with Live Server.
4. Untuk deploy, upload seluruh folder ke Netlify.

File utama:
- index.html
- style.css
- script.js

Folder aset:
- assets/music/lagu.mp3
- assets/images/foto1.jpg sampai foto4.jpg
- assets/animations/lantern.png, sun.png, flower.png
- assets/favorites/ untuk foto makanan/minuman favorit Nadine

Catatan galeri:
- Thumbnail tidak memakai tulisan foto1/foto2/foto3/foto4.
- Foto utama memakai mode no-crop: foto tampil utuh, background memakai blur dari foto yang sama.
- Kalau ingin menambah foto5 dan foto6, masukkan file ke assets/images/foto5.jpg dan assets/images/foto6.jpg, lalu tambahkan datanya di CONFIG.gallery pada script.js.

Catatan musik:
- Musik tidak autoplay sebelum interaksi user karena aturan browser.
- Musik mulai setelah tombol Open Your Birthday Surprise ditekan.

Versi ini mengikuti keputusan terbaru:
- Base tampilan kedua, animasi lebih banyak dan interaktif.
- Floating Lantern Wishes memakai lantern ungu-emas terbaru.
- A Letter From My Heart memakai amplop royal-premium.
- Nadine’s Favorite Things memakai foto asli dan frame akar bunga gold-lavender.
- Pascal dan horse dihapus.
